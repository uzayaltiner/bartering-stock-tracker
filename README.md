# BDO Bartering Stock Tracking Tool

A simple web app for tracking [Black Desert Online](https://www.naeu.playblackdesert.com/) barter item inventory across all 5 tiers.

**Live:** [uzayaltiner.github.io/bartering-stock-tracker](https://uzayaltiner.github.io/bartering-stock-tracker/)

## Features

- 68 barter items with icons from [bdocodex](https://bdocodex.com/) (T1-T5)
- Dark / Light theme
- TR / EN language support
- Tier filtering with multi-select
- Sorting by tier or stock count
- Low stock warnings (red for 0, yellow for 1-10, green for 10+)
- Search across item names in both languages
- Data persists in localStorage
